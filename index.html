Nairobi Metropolitan Real-Time Traffic Update

A real-time traffic monitoring system built using Google Maps API, Apache Kafka, Docker, and Google Colab to simulate a scalable streaming pipeline. The project collects, processes, and visualizes live traffic data across Nairobi's major roads, offering instant insights into congestion hotspots, average speeds, and road conditions. This project showcases my ability to integrate geospatial data with real-time analytics and containerized infrastructure to solve urban mobility challenges.
